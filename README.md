## Prerequisites

-   Python 3.8+
-   pip
-   bs4

## How to Run the Script

-   Install the dependencies
    ```
    python -m pip install -r requirements.txt
    ```
-   Run this on your terminal
    ```
    python stiker.py --site <the line sticker shop page>
    ```
